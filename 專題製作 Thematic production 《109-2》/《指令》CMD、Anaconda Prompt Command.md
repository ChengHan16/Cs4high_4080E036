CMD、Anaconda Prompt Command
============================
指令大小寫皆可

`CD `：切換資料夾位置

`CD\ `：進到根目錄, 不管目前在哪個位置

`CD ..`：回到上一層

`CLS` ：清除螢幕

MD
===
`MD`:建立目錄

`MKDIR`:建立目錄,同MD

`RD`:刪除目錄

`RMDIR`:刪除目錄,同RD 

可在其後加條件   /S      除了目錄本身之外，一併移除指定目錄裡的所有目錄和檔案。
                            /Q      安靜模式，有 /S 時，刪除目錄樹結構不再要求確認。
