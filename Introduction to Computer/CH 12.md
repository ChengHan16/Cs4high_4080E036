# 12-2 　企業中的資訊系統
```
● 資訊系統（information system）是指共同運作來生產資訊的一組硬體、軟體、資料、人員和程序的集合
● 為了協助做出正確可靠的決策，資訊必須具有價值
```
```
● 資訊為了要具有價值，必須要符合：
  ● 正確的
  ● 可驗證的
  ● 及時的
  ● 有組織的
  ● 可取得的
  ● 有用的
  ● 符合成本效益的
```
```
● 典型的大企業是由各個部門、中心和分部所組成，它們統稱為功能單位（functional unit）
  ● 人力資源(HR)
  ● 工程或產品開發
  ● 製造
  ● 行銷
  ● 銷售
  ● 客戶服務
```
```
● 企業資源規劃（enterprise resource planning，ERP）整合 MRP II 與在整個企業流動的資訊流，來管理和協調企業中
  各種持續進行的活動
  ● 產品規劃
  ● 製造和物流
  ● 會計和財務
  ● 銷售
  ● 人力資源
  ● 客戶服務
```
```
● 文件管理系統 (DMS)
  ● 負責儲存和管理公司的文件
  ● 支援存取控制、安全性、文件版本追蹤，以及搜尋功能
● 內容管理系統 (CMS)
  ● 能啟用和管理在網路或web 上各種不同形式的文件和其他檔案（包括媒體和網頁）的發布、修改、組織和存取
  ● 提供內容的安全控制
  ● 使用者透過圖形使用者介面或網頁將內容加入 CMS
```
```
● 交易處理系統（transaction processing system，TPS）是一種用來擷取和處理日常商業活動資料的資訊系統
  ─ 線上交易處理 (OLTP)
```
```
● 管理資訊系統（management information system，MIS）是一種用來產生正確、及時且有組織的資訊，使得經理人和其他
  使用者能進行決策、解決問題、監督活動和追蹤進度的資訊系統。
```
```
● 決策支援系統（decision support system，DSS）幫助使用者分析資訊而進行決策
  ─ 線上分析處理 (OLAP)
```
```
● 專家系統（expert system）是一種用來擷取和儲存人類專家的知識，然後模擬人類的推理和決策過程的資訊系統
```
# 12-3 　資訊產業中的職業
```
● 各種規模的企業和政府機關都會用到各式電腦、行動裝置和其他科技產品
● 大企業常使用電腦和其他科技來回應和轉接電話、處理訂單、更新庫存，以及管理公司財務等
  ● 管理
  ● 研發及軟體開發
  ● 技術支援服務
  ● 系統營運
  ● 教育訓練與技術支援
  ● 資訊安全
  ● 網路行銷與策略
```
```
● 資訊設備領域包含電腦、行動裝置與相關硬體（如磁碟機、光碟機、螢幕、印表機，以及通訊和網路裝置等）的製造商和經銷商
● 軟體和 App 領域包含專門開發、製造和支援電腦、web 和行動裝置上各種軟體和 App 的公司
```
```
● 資訊服務及維修領域是替顧客定期保養、安裝元件和維修服務
```
```
● 資訊產品銷售人員必須具備一般的科技知識，同時對自己銷售的產品更要充分瞭解
```
```
● 包括各級學校、在職訓練機構及私人企業，都需要具備資格的教育人員，能提供資訊相關的教育及訓練
● 企業的教育訓練講師（trainer）教導員工如何使用軟體和 App、設計和開發系統、撰寫程式、整合和同步來自多個裝置上 App 
  的資料，以及執行其他資訊相關活動
```
```
● 資訊服務專員（help desk specialist）親自或透過電話、電子郵件或討論區，來回答關於硬體、軟體與網路方面的問題
● IT 顧問（IT consultant）通常是因為具備客戶所需的專業經驗，能提供客戶所需的技術諮詢服務而被聘用
```

# 科技最前線 12-2：開發Mobile App
```
● 開發 mobile app 的3種方式：
  ─ 原生app（native app）：針對執行特定作業系統的行動裝置所設計
  ─ Mobile web app：其實是網站，不是發布到 app store，而是部署到某個網站伺服器，讓人們經由行動瀏覽器使用
  ─ 混合式（hybrid）app：結合原生 app和 mobile web app的特點
```

# 科技最前線 12-3：無人機
```
● 科技最前線 12-3：無人機
  無人機（drone）的專業術語叫做無人飛行載具（UAV）或無人飛行系統（UAS）
 
● 它是由包括機載電腦加 GPS、遙控裝置，可能還有在電腦或行動裝置上的 App 共同運作的一種飛行器

  ─ 自動駕駛功能
  ─ 高解析度相機
```
