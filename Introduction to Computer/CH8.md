# 8-1 　儲存體
```
● 儲存媒體（storage medium）也稱為輔助儲存體（secondary storage），它是科技產品用來保存資料、資訊、
程式與應用軟體的實體材質。

● 雲端寄存空間是另一種儲存選擇，它是將資料儲存在 Internet 的伺服器上
```
```
● 儲存裝置（storage device）是用來將資料記錄到儲存媒體，或從儲存媒體擷取出資料的一種硬體

● 寫入（writing）是將資料、指令與資訊從記憶體轉移到儲存媒體的過程。

● 讀取（reading）則是從儲存媒體將這些項目傳輸到記憶體的過程
```
```

# 8-2 　硬碟機
```
● 硬碟機（hard drive）這個術語被用來統稱硬碟（hard disk）和 SSD。

硬碟機可以是內接式（internal）或外接式（external）；
它們可能是常駐在電腦或行動裝置的內部元件， 或者是可連接到某台電腦或行動裝置的外部裝置。
```                                        
```
硬碟的儲存容量是由：
─ 內含的磁盤個數
─ 塗在磁盤表面上的磁性物質成分
─ 使用的錄寫技術（縱向或垂直）
─ 密度
```
```
● 格式化（formatting）是將磁碟分割成多個磁軌和磁區的過程
```
```
● 硬碟的特性包含：

┌────────────┐   ┌────────────┐   ┌────────────┐
     磁軌             磁區              磁盤
└────────────┘   └────────────┘   └────────────┘
┌────────────┐   ┌────────────┐   ┌────────────┐
   磁盤尺寸           讀寫頭          每分鐘轉速
└────────────┘   └────────────┘   └────────────┘
```
```
● 硬碟的運作原理

步驟一
電路板控制讀寫頭驅動器和小型馬達的移動動做。

步驟二
電腦開機運作期間，小型馬達一直旋轉磁盤。

步驟三
當軟體要求存取磁碟時，讀寫頭找出資料的目前位置或新位置。

步驟四
讀寫頭驅動器將讀寫頭機械臂移至磁盤上的正確位置，準備讀寫資料。
```
```
● 讀寫頭如果碰觸到磁盤表面就會造成讀寫頭損毀（head crash）
● 定時備份硬碟內容是絕對必要的
```
```
● 固態硬碟（solid-state drive，SSD）是一種內含處理器來管理儲存空間的快閃記憶體儲存裝置。
● 以下是 SSD 勝過傳統（磁性）硬碟的優點：

  存取時間較快      傳輸速率較快       運作更安靜         更耐用           重量更輕         
└────────────┘   └────────────┘   └────────────┘   └────────────┘   └────────────┘  
        ┌────────────┐   ┌────────────┐   ┌────────────┐   ┌────────────┐
           耗電量較少      產生的熱能更少       壽命更長     不需要進行磁碟重組
        └────────────┘   └────────────┘   └────────────┘   └────────────┘
```
```
● 外接式硬碟機（external hard drive）是一個分離的獨立儲存裝置，透過傳輸纜線連接到電腦或行動裝置的 USB 或其他埠
```
```
● 這種由兩個或更多硬碟機整合在一起的方式稱為 RAID （redundant array of independent disks，磁碟陣列）
```
# 8-3 　 可攜式快閃記憶體儲存體
```
● 記憶卡（memory card）是一種卸除式快閃記憶體儲存裝置，
通常高度或寬度不會大於 1.5 吋，使用時插入電腦、行動裝置或讀寫卡機的插槽中。
┌────────────┐   ┌────────────┐   ┌────────────┐   ┌────────────┐   ┌────────────┐  
     SDHC             SDXC             miniSD         microSDHC        microSDXC
└────────────┘   └────────────┘   └────────────┘   └────────────┘   └────────────┘  
   ┌────────────┐      ┌────────────┐      ┌────────────┐       ┌────────────┐
         CF            xD Picture Card    Memory Stick PRO Duo        M2
   └────────────┘      └────────────┘      └────────────┘       └────────────┘
```
```
很多類型的電腦和裝置都配備給記憶卡使用的插槽。
(例：數位相機、讀寫卡機、智慧型手機、印表機、筆記型電腦、平板電腦)
```
```
SD卡的運作原理
步驟一
當記憶卡插入讀寫卡機或卡片插槽，記憶卡的金屬導片會接觸到讀寫裝置的連接端，卡片便可以與讀寫裝置進行資料的傳輸動作。

步驟二
記憶卡上有一邊會有凹槽，用來防止記憶卡意外滑出讀寫卡機或插槽。

步驟三
快閃記憶體品片儲存著資料。
當接收到要求時，控制晶片會將儲存在快閃記憶品片上的資料傳輸到金屬導片上，而且必要時使用暫存器當作暫存空間。
      
步驟四
有些記憶卡附有防寫開關，以防止使用者不小心刪除了儲存在快閃記憶體晶片中的資料。
```
```

是一種快閃記憶體儲存裝置，使用方式是插入電腦或行動裝置上的 USB 埠。

# 8-5 　光碟
```
● 光碟（optical disc）
是一種儲存媒體，它是一片可由雷射光進行讀寫，由金屬、塑膠與塗漆材質製成，且表面平坦方便攜帶的圓盤。
```
```
雷射光如何讀取光碟上的資料
步驟1：雷射二極體(laser diode)向光碟發出一道光束。

步驟2如果光束打到凹洞，光線會散射。如果光東打到平面區域，便會反射回雷射二極體。

步驟3：反射回來的光束會被轉向到感光二極體，而由它傳送數位訊號1給電腦。如果沒有反射光束，則解讀為數位訊號0。
```
```
● 光碟通常只有一軌，從光碟中心呈螺旋狀向外環繞至光碟邊緣
 這個單一光碟軌道也會平均分割成大小相同的多個光碟磁區，用來儲存資料
```

● DVD-ROM（DVD-read-only memory）是一種高密度光碟，使用者可讀取但不能寫入或消除內容
● DVD-R 與 DVD+R 是兩種彼此不相容的 DVD可錄寫 WORM 格式，使用者只能寫入一次但不能消除內容
● DVD-RW、DVD+RW 和 DVD+RAM 是彼此不相容的可重寫 DVD 格式，使用者可重複寫入多次
