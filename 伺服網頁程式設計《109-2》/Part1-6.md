

## 虛擬環境設定
> 建立：conda create --name myflask01 python=3.8 <br>
> 啟動：conda activate myflask01 <br>
> 關閉：conda deactivate

## 安裝套件
> pip install flask (part1)<br>
> pip install flask_wtf <br>
> pip install email_validator <br>
> pip install flask-sqlalchemy <br>
